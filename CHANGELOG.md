# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog and adapted for the D3Bet workflow.

## [Unreleased]

### Added

- server to client architecture with `ASP.NET Core`, `Identity`, and `OpenIddict`
- WPF operator client connected through authenticated API calls
- local network server discovery for the desktop client
- customer display mode for kiosk and public screens
- audit trail with persistent audit history
- admin audit tab in the WPF client
- realtime synchronization with `SignalR`
- role separation for `Admin` and `Operator`

### Improved

- marketing-oriented repository profile and bilingual README
- friendlier error handling and retry UX in the desktop client
- structured server diagnostics with correlation and trace identifiers
